# ☕ Coffee-E Website

## 📌 Project Overview

**Coffee-E** is a simple and responsive coffee shop website created using **HTML and CSS**.
The website presents a modern coffee shop interface where users can explore the menu, view testimonials, browse the gallery, and contact the shop.

The goal of this project is to practice **frontend web development** and build a clean responsive layout for a coffee shop.

---

## 🚀 Features

* Responsive navigation bar
* Home hero section
* About Us section
* Menu section with coffee and food items
* Customer testimonials
* Image gallery
* Contact information
* Contact form
* Social media icons

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Google Fonts**
* **Font Awesome Icons**

## 📸 Website Sections

### 🏠 Home

Displays the hero section with a welcoming message and call-to-action buttons.

### ℹ️ About Us

Provides information about the coffee shop and its mission.

### ☕ Menu

Shows different beverages and food items available.

### ⭐ Testimonials

Displays feedback from customers.

### 🖼 Gallery

Shows images related to the coffee shop.

### 📞 Contact

Includes:

* Address
* Email
* Phone number
* Working hours
* Contact form
